# Learning-center-analysis
This Power BI dashboard analyzing enrollment, fees, courses, gender and branches in Learning Center. 
This Power BI report transforms raw student data (from Excel) into actionable business insights for a learning center. It helps answer questions like:

- What is the total/average course fee revenue?
- How many students are enrolled, and what is their average age & course duration?
- Which courses are most popular? 
- How does enrollment and gender distribution vary across branches? 
- Which cities/regions generate the highest fees? 

Key KPIs (from the dashboard):
- Total Average Course Fee: $191,667
- Average Student Age: 28.10
- Total Number of Students: 150
- Average Course Duration: 55.80 

## Features

- KPI Cards — instant view of total fees, student count, average age, and duration
- Donut Chart — student distribution by course 
- Line Chart — average course fee trend by month 
- Bar Charts — number of students by branch & gender 
- Horizontal Bar — course fee comparison by city 
- Dark modern theme — clean, professional look with easy filtering

## Technologies Used

- Power BI Desktop
- Power Query — data cleaning & transformation
- DAX — calculated measures 
- Data Source: Excel file with student records
